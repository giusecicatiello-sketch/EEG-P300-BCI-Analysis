# EEG-P300-BCI Analysis

## Overview

This repository contains a MATLAB pipeline for the analysis of EEG signals in Brain-Computer Interface (BCI) applications based on the P300 Event-Related Potential.

The project includes preprocessing, ERP extraction, feature extraction and machine learning classification.

---

## Features

- EEG preprocessing
- Signal filtering
- ERP extraction
- P300 analysis
- Feature extraction
- Machine Learning classification
- MATLAB implementation
